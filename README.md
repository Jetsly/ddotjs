# ddot

Lightweight tools framework 

---

## Getting started

### Install
```bash
$ git clone https://github.com/Jetsly/ddot.git
```
###  Build
```bash
$ npm run build
```
### Lint
```bash
$ npm run lint
```
### Test
```bash
$ npm test
```
