### API Reference

```js
import { 
    is,
    com
} from 'ddot'
```

#### `is`
```js
$ is.func // is function
$ is.num  // is number
$ is.str  // is string
$ is.obj  // is object
$ is.arr  // is array
```

### `lo`
```js
$ lo.query   // parse url search to obj
$ lo.cookie  // get and set cookie
```
